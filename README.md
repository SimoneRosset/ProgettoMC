# Crossy Cloud 
## Description: 
Crossy Cloud is a vertical scroll endless game. Drag your red balloon's rope to avoid hitting birds and reach the sky. Be careful! The higher you go, the more difficult it will become! 

## Language & API: 
Lua, Corona API

## IDE: 
Corona SDK
